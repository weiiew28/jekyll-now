Von Neumann MinMax theorem

Let P⊂RmP\subset \mathbb{R}^{m}P⊂Rm and Q⊂RnQ\subset \mathbb{R}^{n}Q⊂Rn be convex compact sets, and let ℓ(p,q):Rm×Rn→R\ell (p,q):\mathbb{R}^{m}\times \mathbb{R}^{n}\to \mathbb{R}ℓ(p,q):Rm×Rn→R be a bilinear scalar function. Then,
max⁡p∈Pmin⁡q∈Qℓ(p,q)=min⁡q∈Qmax⁡p∈Pℓ(p,q)\max_{p\in P}\min_{q\in Q} \ell(p,q) = \min_{q \in Q}\max_{p\in P}\ell(p,q)maxp∈P​minq∈Q​ℓ(p,q)=minq∈Q​maxp∈P​ℓ(p,q)

This is the theorem for the zero-sum game, which says that given the full information setting, where the profit/loss is known in advance, there exists an equilibrium strategy pair (p∗,q∗)(p_*,q_*)(p∗​,q∗​) such that no player wants to switch to other strategies.

Note that strategy for the first and second player are not known in advance, and they reveal their strategy simultaneously when the game starts.

To understand the theorem more concretely, let us call the first player the row player, and he can choose from {1,⋯,n}\left\{1,\cdots,n\right\}{1,⋯,n} as his strategy. A mixed strategy is also allowed, where one can choose strategy iii with certain probablity (pip_ipi​). The probability simplex PPP characterizes his mixed strategy selection space. Similarly, the second player, which we call the column player, can choose from {1,⋯,m}\left\{1,\cdots, m\right\}{1,⋯,m} as his strategy. The probablity simplex QQQ characterizes his mixed strategy selection space.

Given strategy iii the row player has picked and strategy jjj the columns player has picked, it incurs a loss for the row player (or equivalently, a gain for the column player), which we denote Ai,jA_{i,j}Ai,j​. AAA is a m×nm\times nm×n matrix. In the zero sum game, one player’s loss is another player’s gain. Row player wants to minimize the loss while the column player wants to maximize the gain (or the loss of the row player).

One might ask why we want a mixed strategy, not pure strategy? Suppose no mix strategy is allowed, then it is possible that the equilibruim does not exists at all, illustrated by the following toy examples:

Suppose we have the following loss matrix (222 strategies for the row and column players)

(2311.5)\begin{pmatrix} 2 & 3\\ 1 & 1.5 \end{pmatrix}(21​31.5​)

For the row player, if he picks strategy 111, the row player will pick strategy 222. If he picks strategy 222, the row player will still pick strategy 222 so that he will suffer a greater loss 1.5. Thus the column player will pick strategy 222 regardless of strategy of the row player. Having that said, the row player will then pick strategy 222 to incur less loss. For this loss matrix, the pair (2,2)(2,2)(2,2) is an equilibrium strategy pair.

Now let us change the loss matrix a little bit:

(1321)\begin{pmatrix} 1 & 3\\ 2&1 \end{pmatrix}(12​31​)

It is easy to check that no equilibrium pair exists. For instance, let us look at the pair (1,1)(1,1)(1,1). Fixing row player’s strategy, the column player has incentive to switch to strategy 222 since it makes the loss for the row player larger. Similar arguments hold for all pairs.

In contrast, Von Neumann’s theorem says that if we allow mixed strategy, then an equilibrium mixed strategy pair always exists! In this case, the expected loss for the row player under the mixed strategy pair (p,q)(p,q)(p,q) is

∑i=1m∑j=1npiAi,jqj=pTAq\sum_{i=1}^{m}\sum_{j=1}^{n} p_iA_{i,j}q_j=p^{T}Aq∑i=1m​∑j=1n​pi​Ai,j​qj​=pTAq

Observe that pTAqp^{T}AqpTAq is bilinear in ppp and qqq and the theorem can be applied directly. Indeed, we find that the equilibrium mixed strategy for the row player is (13,23)(\frac{1}{3},\frac{2}{3})(31​,32​), and the equilibrium mixed strategy for the column player is (23,13)(\frac{2}{3},\frac{1}{3})(32​,31​).

Given the loss matrix AAA, is it also computationally efficient to find the equilibrium pair?

Blackwell’s generalization

In the Von Neumann minmax theorem, the loss is a pure scalar for a pair of strategy. Blackwell generalized the scalar to vector loss. One can imagine the outcome is charactered by several dimensions, and there is some tradeoff between these factors.

For the zero sum game, the outcome is reduced to a scalar because the sum is 0 and the equilibrium are the saddle points in the space of P×QP\times QP×Q. In the vector loss case, we generalize it to a target set SSS.

We define SSS to be approachable if there exists a strategy for the column player {qt}t=1∞\left\{q_t\right\}_{t=1}^{\infty}{qt​}t=1∞​ in the repeated simultaneous game that guarantees

lim⁡T→∞d(LT,S)=0∀{pt}t=1∞\lim_{T\to \infty} d(L_{T},S)=0 \;\; \forall \left\{p_t\right\}_{t=1}^{\infty}limT→∞​d(LT​,S)=0∀{pt​}t=1∞​

where
LT=1T∑t=1Tℓ(pt,qt)L_{T}=\frac{1}{T}\sum_{t=1}^{T}\ell(p_t,q_t)LT​=T1​∑t=1T​ℓ(pt​,qt​)

Example of vector loss

In the expert’s problem, we have a mixed strategy qtq_tqt​ for picking the experts in each round ttt, and incurs a loss with respect to each expert:

(qt⋅pt−pt,1,⋯.qt⋅pt−pt,n)(q_t\cdot p_t - p_{t,1}, \cdots. q_t\cdot p_t -p_{t,n})(qt​⋅pt​−pt,1​,⋯.qt​⋅pt​−pt,n​)

We want lim⁡T→∞1T∑t=1Tqt⋅pt−pt,j≤0\lim_{T\to \infty} \frac{1}{T}\sum_{t=1}^{T}q_t\cdot p_t-p_{t,j}\leq 0limT→∞​T1​∑t=1T​qt​⋅pt​−pt,j​≤0 for all jjj such that the performance is similar to the best expert in hindsight. Here, our target set SSS is the negative orthant.

Theorem

The main theorem is that when PPP,QQQ, SSS are convex and compact, the following are equivalent:

  1. SSS is approachable
  2. Every half space that contains SSS is approachable
  3. For every ppp, there exists qqq such that ℓ(p,q)∈S\ell(p,q) \in Sℓ(p,q)∈S

Using the theorem, it immedaitely implies the existence of the online strategy for the expert’s problem such that one can be as good as the best expert.

The proof of the theorem is very interesting and it builds upon the Von Neuman’s minmax theorem. Half space also plays a key role here for using the Von Neuman’s minmax theorem. [I feel that the intuition behind the proof is that every compact convex set is an intersection of halfplane.]
