Proximal Method

General proximal method itself is a very interesting topic that worths an entire post for discussion. A concise description is that in each iteration of the minimization, (1) we want to find a ‘nice’ upper bound function that is easy to optimize (2) minimizing the upper bound (exact or inexact) yields the next iterate.

We will briefly discuss two simple examples for illustration. First let us consider the standard gradient descent method for a function f:Rn→Rf:\mathbb{R}^{n}\to \mathbb{R}f:Rn→R whose gradient is Lipschitz:

∣∇f(x)−∇f(y)∣≤L∣x−y∣|\nabla f(x)-\nabla f(y)|\leq L|x-y|∣∇f(x)−∇f(y)∣≤L∣x−y∣ (1)

For the t+1t+1t+1-th iterate,

xt+1=xt−γt∇f(xt)x_{t+1}=x_{t}-\gamma_t \nabla f(x_t)xt+1​=xt​−γt​∇f(xt​)

where γt=1L\gamma_t = \frac{1}{L}γt​=L1​ is the step size. Equivalently, xt+1x_{t+1}xt+1​ is the solution to the following convex function:

xt+1=argminyf(xt)+<∇f(xt),y−x>+L2∣∣y−xt∣∣2x_{t+1} = argmin_{y} f(x_t)+<\nabla f(x_t),y-x>+\frac{L}{2} ||y-x_t||^2xt+1​=argminy​f(xt​)+<∇f(xt​),y−x>+2L​∣∣y−xt​∣∣2.

Indeed, by the Lipschitz condition (1), one can show that

f(y)≤f(x)+<∇f(x),y−x>+L2∣∣y−x∣∣2:=fx(y)∀yf(y)\leq f(x)+<\nabla f(x),y-x>+\frac{L}{2}||y-x||^2: = f_{x}(y) \;\; \forall yf(y)≤f(x)+<∇f(x),y−x>+2L​∣∣y−x∣∣2:=fx​(y)∀y

Therefore, one can view gradient descent as a special case of the proximal method with the upper bound fx(y)f_x(y)fx​(y).

Proximal-linear

Now we will talk about how the authors adapt the proximal method to solve a particular structral composite optimization problem of the form:

min⁡h(x)=f(g(x))\min h(x)= f(g(x))minh(x)=f(g(x))

where g:Rn→Rmg:\mathbb{R}^{n}\to \mathbb{R}^{m}g:Rn→Rm is smooth and f:R→Rf:\mathbb{R}\to \mathbb{R}f:R→R is convex. For this composite problem, it is not necessary convex in the variable, and the optimal solution is not necessarily unique either. Let SSS be the set of all optimal solutions.

They

  1. first use the linear appoximate of ggg, and approximate h(y)h(y)h(y) by hx(y)=f(g(x)+<∇g(x),y−x>)h_{x}(y)=f(g(x)+<\nabla g(x),y-x>)hx​(y)=f(g(x)+<∇g(x),y−x>)
  2. then try to solve hx(y)+L2∣∣y−x∣∣2h_{x}(y)+\frac{L}{2}||y-x||^2hx​(y)+2L​∣∣y−x∣∣2. (Observe that for this step, fff is convex, therefore, the solution is unique.).

Conditions assumed:

  1. (sharpness) ∣h(x)−h(x∗)∣≥λ∣∣x1−x∗∣∣2|h(x)-h(x_*)|\geq \lambda ||x_1-x_*||^2∣h(x)−h(x∗​)∣≥λ∣∣x1​−x∗​∣∣2 for all x∗∈Sx_*\in Sx∗​∈S
  2. (close approximation) ∣h(y)−hx(y)∣≤L2∣x−y∣2|h(y)-h_{x}(y)|\leq \frac{L}{2}|x-y|^2∣h(y)−hx​(y)∣≤2L​∣x−y∣2

By the second step of the algorithm and the close approximation property, we immediately have:

h(xt+1)≤hxt(xt+1)+L2∣∣xt+1−xt∣∣2≤h(xt)h(x_{t+1}) \leq h_{x_t}(x_{t+1})+\frac{L}{2}||x_{t+1}-x_t||^2 \leq h(x_t)h(xt+1​)≤hxt​​(xt+1​)+2L​∣∣xt+1​−xt​∣∣2≤h(xt​)

where the second inequality holds because xt+1x_{t+1}xt+1​ is the minimizer of hxt(y)+L2∣∣y−xt∣∣2h_{x_t}(y)+\frac{L}{2}||y-x_t||^2hxt​​(y)+2L​∣∣y−xt​∣∣2. Hence, the above algorithm enjoys the monotonicity property in which the objective value is non-increasing.

Moreover, we can use the close approximation condition and strong conveixty of hx(y)+L2∣∣y−x∣∣2h_x(y)+\frac{L}{2}||y-x||^2hx​(y)+2L​∣∣y−x∣∣2 to get:

hxt(xt+1)+L2∣∣xt+1−xt∣∣2≤hxt(x∗)+L2∣∣x∗−xt∣∣2−L2∣∣x∗−xt+1∣∣2h_{x_t}(x_{t+1})+\frac{L}{2} ||x_{t+1}-x_{t}||^2 \leq h_{x_t}(x_*)+\frac{L}{2} ||x_*-x_{t}||^2-\frac{L}{2}||x_{*}-x_{t+1}||^2hxt​​(xt+1​)+2L​∣∣xt+1​−xt​∣∣2≤hxt​​(x∗​)+2L​∣∣x∗​−xt​∣∣2−2L​∣∣x∗​−xt+1​∣∣2

and

hxt(x∗)≤h(x∗)+L2∣∣xt−x∗∣∣2h_{x_t}(x_*) \leq h(x_*)+\frac{L}{2}||x_t-x_*||^2hxt​​(x∗​)≤h(x∗​)+2L​∣∣xt​−x∗​∣∣2

It follows that:

h(xt+1)−h(x∗)+L2∣∣x∗−xt+1∣∣2≤L∣∣x∗−xt∣∣2h(x_{t+1})-h(x_*)+\frac{L}{2}||x_*-x_{t+1}||^2 \leq L||x_*-x_t||^2h(xt+1​)−h(x∗​)+2L​∣∣x∗​−xt+1​∣∣2≤L∣∣x∗​−xt​∣∣2 (3)

Finally, using the sharpness condition,

(λ+L2)∣∣xt+1−x∗∣∣2≤L∣∣x∗−xt∣∣2(\lambda+\frac{L}{2}) ||x_{t+1}-x_*||^2 \leq L||x_*-x_t||^2(λ+2L​)∣∣xt+1​−x∗​∣∣2≤L∣∣x∗​−xt​∣∣2 for all x∈Sx\in Sx∈S

Phase Retrieval revisit

Now let us consider the problem of interest: supposer we observe some measurements:

bi=<ai,x>2b_i = <a_i, x>^2bi​=<ai​,x>2, i=1,⋯,mi=1,\cdots, mi=1,⋯,m,

can we recover xxx and what is the sample complexity? The authors consider the following non-convex formulation:

min⁡1m∑i=1m∣bi−<ai,x>2∣1\min \frac{1}{m}\sum_{i=1}^{m} |b_i -<a_i, x>^2|_{1}minm1​∑i=1m​∣bi​−<ai​,x>2∣1​

Note that the penalty function is the ℓ1\ell_1ℓ1​ norm (why not ℓ2\ell_2ℓ2​?). Another observation is that the optimal solution is invariant under sign flip.

In the language of the proximal linear algorithm, fff here is a convex functon that maps Rm\mathbb{R}^{m}Rm to R\mathbb{R}R, given by:

f(y1,⋯,ym)=∑i=1m∣yi∣=∣∣y∣∣1f(y_1,\cdots,y_m)=\sum_{i=1}^{m} |y_i| = ||y||_1f(y1​,⋯,ym​)=∑i=1m​∣yi​∣=∣∣y∣∣1​

and ggg here is a smooth funciton that maps Rn→Rm\mathbb{R}^{n} \to \mathbb{R}^{m}Rn→Rm, whose iii-th coordinate is given by:

g(x)i=1m(<ai,x>2−bi)g(x)_i = \frac{1}{m} (<a_i,x>^2- b_i)g(x)i​=m1​(<ai​,x>2−bi​).

For the phase retrieval problem, we have some stability conditon (Eldar and Mendelson) as follows:

∣∣(Ax)2−(Ay)2∣∣1≥λ∣∣x−y∣∣⋅∣∣x+y∣∣||(Ax)^2-(Ay)^2||_{1} \geq \lambda ||x-y||\cdot ||x+y||∣∣(Ax)2−(Ay)2∣∣1​≥λ∣∣x−y∣∣⋅∣∣x+y∣∣ for all x,y∈Rnx,y\in \mathbb{R}^{n}x,y∈Rn (4)

(Note that this means that if the solution is not the original signal, then the function value will grow fast!)

Hence,

∣h(x)−h(x∗)∣=1m∣(Ax)2−(Ax∗)2∣≥λ∣∣x−x∗∣∣⋅∣∣x+x∗∣∣|h(x)-h(x_*)| = \frac{1}{m} |(Ax)^2-(Ax_*)^2| \geq \lambda ||x-x_*|| \cdot ||x+x_*||∣h(x)−h(x∗​)∣=m1​∣(Ax)2−(Ax∗​)2∣≥λ∣∣x−x∗​∣∣⋅∣∣x+x∗​∣∣

Note that

1m∣(Ax)2−(Ax∗)2∣=1m∣(x−x∗)ATA(x+x∗)∣\frac{1}{m} |(Ax)^2-(Ax_*)^2| = \frac{1}{m} |(x-x_*)A^{T}A (x+x_*)|m1​∣(Ax)2−(Ax∗​)2∣=m1​∣(x−x∗​)ATA(x+x∗​)∣

The stability condition is equivalent to having a lower bound on the random matrix 1m∑i=1maiaiT\frac{1}{m} \sum_{i=1}^{m} a_{i}a_{i}^{T}m1​∑i=1m​ai​aiT​.

For the close approximation condition, we have that

hx(y)=1m∑i=1m∣<ai,x>2−bi+2<ai,y−x><ai,x>∣h_{x}(y) = \frac{1}{m}\sum_{i=1}^{m}|<a_i, x>^2-b_i + 2<a_i,y-x> <a_i, x>|hx​(y)=m1​∑i=1m​∣<ai​,x>2−bi​+2<ai​,y−x><ai​,x>∣

By the triangle inequality, it follows that
∣hx(y)−h(y)∣≤1m∑i=1m<ai,x−y>2|h_x(y)-h(y)|\leq \frac{1}{m}\sum_{i=1}^{m} <a_i,x-y>^2∣hx​(y)−h(y)∣≤m1​∑i=1m​<ai​,x−y>2 (5)
